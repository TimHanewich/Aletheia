Insider Trading

Read and monitor insider transactions for any publically traded company.

Search People

Search the Aletheia database for an insider across all publicly traded companies.

GET
https://aletheia.azurewebsites.net/api/SearchPeople

Parameters

Required
term - The search term

Example Requests

https://aletheia.azurewebsites.net/api/SearchPeople?term=smith
https://aletheia.azurewebsites.net/api/SearchPeople?term=musk
https://aletheia.azurewebsites.net/api/SearchPeople?term=ellison

Get Latest Transactions

Download the latest insider transactions for a person or company.

GET
https://aletheia.azurewebsites.net/api/LatestTransactions

Parameters

Required
entity - The entity type you are searching through. 0 = Company, 1 = Person.
Required
cik - The CIK of either the company or person you are requesting latest transactions for.
  • If you are requesting latest transactions for a company, you can supply the trading symbol instead.
Optional
top - How many records you would like returned. Default is 20.
Optional
before - Filter to transactions only before a particular date.
  • Date supplied in format YYYYMMDD. For example, "20191231".
Optional
securitytype - Filter security type. 0 = non-derivative (equity), 1 = derivative.
Optional
transactiontype - Filter to a particular transaction type.
  • 0 = purchase
  • 1 = sale
  • 2 = transaction voluntarily reported earlier than required
  • 3 = grant or award
  • 4 = sale back to issuer
  • 5 = payment of excercise price or tax liability
  • 6 = discretionary transaction
  • 7 = excercise or conversion of derivative security
  • 8 = conversion of derivative security
  • 9 = expiration of short derivative position
  • 10 = expiration of long derivative position
  • 11 = excercise of out-of-the-money derivative
  • 12 = excercise of in-the-money derivative
  • 13 = bona fide gift
  • 14 = small acquisition
  • 15 = acquisition or disposition by will or laws
  • 16 = deposit into or withdrawal from voting trust
  • 17 = other acquisition or disposition
  • 18 = equity swap
  • 19 = disposition due to tender of shares

Example Requests

https://aletheia.azurewebsites.net/api/LatestTransactions?entity=0&cik=msft&top=10&before=20190101&securitytype=0&transactiontype=0
https://aletheia.azurewebsites.net/api/LatestTransactions?entity=0&cik=0000789019&top=70&securitytype=1
https://aletheia.azurewebsites.net/api/LatestTransactions?entity=1&cik=0001576843
